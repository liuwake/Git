## 备注

- 当VisualStudio.gitignore 无法在explorer中重命名为.gitignore的时候,可以按住SHIFT,右键单击文件夹空白处,```Open PowerShell window here```使用powershell命令如
```rename-item .\VisualStudio.gitignore -newname .gitignore```
